# Upload to GitHub Pages (`SR-launcher`)

Site URL: https://kotipipa7-debug.github.io/SR-launcher/

## What to upload

Upload **everything** inside this folder to the root of
`https://github.com/kotipipa7-debug/SR-launcher` (branch `main`):

- `index.html`
- `privacy.html`
- `terms.html`
- `styles.css`
- `assets/sb-logo.png`
- `downloads/SB-Launcher-Setup-2.1.0.exe` (~94 MB — required for on-site download)

## Steps (browser)

1. Open the repo → **Add file → Upload files**
2. Drag all files/folders from `docs/site`
3. Commit to `main`
4. Confirm **Settings → Pages → Deploy from a branch → main / (root)**

After 1–2 minutes, open the site and click **Download for Windows**.

## Optional: GitHub Release

Create a Release named `v2.1.0` and attach `SB-Launcher-Setup-2.1.0.exe`.
The homepage automatically prefers the latest Release asset when available.
